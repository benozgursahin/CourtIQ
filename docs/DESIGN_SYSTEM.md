# CourtIQ Design System v1.0

CourtIQ, Garmin Fenix 8 Pro 47 mm için tasarlanan; tenis, aktif yaşam ve profesyonel günlük kullanım odaklı premium performans saat yüzüdür.

## Design Principle

Tek bakışta doğru karar.

## Target User

- Haftada 3–5 gün spor yapan aktif profesyonel
- Orta/ileri seviye tenis oyuncusu
- Sağlık ve performans verilerini takip eden kullanıcı
- Minimal ama zengin veri sunumu isteyen Garmin kullanıcısı

## Visual Identity

- Background: AMOLED Black
- Primary Accent: Fenix Pro Orange
- Main Text: White
- Ready / Energy: Green
- Recovery: Purple
- Training: Blue
- Heart Rate: Red

## Information Hierarchy

1. Time
2. Court Status
3. Energy
4. Health
5. Environment
6. Details

## Court Status Labels

- READY
- GOOD
- EASY DAY
- RECOVER
- REST

## Core Metrics

- Time
- Date
- Battery
- Heart Rate
- Steps
- Body Battery
- Training Readiness
- Recovery Time
- HRV Status
- Weather

## Always-On Display Rules

- No animation
- No seconds
- Reduced color intensity
- Thin ring only
- Maximum readability
