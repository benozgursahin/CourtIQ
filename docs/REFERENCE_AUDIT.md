# CourtIQ Reference Audit v0.1

## Base Reference

Garmin official Connect IQ sample:

- Repository: garmin/connectiq-apps
- Sample: watchfaces/TypedFace

## Why We Use It

- Official Garmin reference
- Valid watch face structure
- Existing complication model
- Existing resources and drawables
- Good baseline for learning Garmin's intended architecture

## What We Keep

- Manifest structure
- Resource organization
- Basic watch face lifecycle
- Complication idea
- Battery / date / heart / steps examples

## What We Replace

- Visual design
- Layout system
- Naming
- Typography
- Color system
- Data hierarchy
- CourtIQ performance logic

## Product Direction

CourtIQ will not be a copy of TypedFace.  
TypedFace is the technical reference.  
CourtIQ is the product layer.
